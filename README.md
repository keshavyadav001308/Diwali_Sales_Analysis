 Diwali Sales Analysis

 Project Description
Diwali Sales Analysis is a data analytics project that performs **Exploratory Data Analysis (EDA)** on festive sales data to understand **customer behavior, purchasing patterns, and sales trends** during the Diwali season. The project aims to extract **business-driven insights** that can help organizations improve marketing strategies and maximize festive revenue.

 Objectives
- Analyze customer demographics influencing sales
- Identify high-performing product categories
- Study state-wise and occupation-wise sales trends
- Understand spending behavior during the festive season

Dataset Information
- Dataset: Diwali Sales Dataset
- Contains information related to:
  - Customer age, gender, marital status
  - State and occupation
  - Product categories
  - Sales amount

Technologies Used
- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Jupyter Notebook / Google Colab**

Analysis Performed
- Data cleaning and preprocessing
- Handling missing and inconsistent values
- Exploratory Data Analysis (EDA)
- Data visualization using charts and graphs
- Customer segmentation analysis

 Key Insights
- Certain states and product categories contribute significantly to Diwali sales
- Married customers and specific age groups show higher purchasing trends
- Occupation and gender play a crucial role in spending behavior

Business Impact
This analysis helps businesses to:
- Improve targeted festive marketing campaigns
- Optimize inventory planning
- Enhance customer segmentation
- Increase overall festive-season profitability
